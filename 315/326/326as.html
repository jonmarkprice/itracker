CS 315

Assignment for 26 March 2015

This homework problem is to be done individually, not in pairs or other groups. It is due by noon Thursday, 26 March.

  1. Define a database table with three or four columns of your choosing. For example, you might have lastName, firstName and emailAddress, and of course a primary key field.
  2. Write a PHP program or HTML page that presents the user with a form that allows them to enter data to store in the table. There may or may not be missing data, as appropriate for your specific columns. Use form controls to prevent errors as much as possible.
  3. Write a different PHP program that allows the user to request some (or all) of the data from the table and that presents it in some appropriate manner, in a way that makes sense. You should present the results sorted in a useful way, and it would be especially nice if the user could choose how it was sorted.
  4. The SQL statement that creates your table should be in a file named yourlastname.sql, where “yourlastname” is your last name in ALL LOWERCASE LETTERS. The file should be one that you have written directly with an editor, not one generated by a tool such as mysqldump. You may choose to pre-populate the table with some data to start with, in which case you should put SQL insert statements in this same .sql file.
  5. Make sure there is a .phps file for each of your .php files.

By the due time and date, send an email message addressed to the homework dropbox 315@borax.truman.edu with a subject of “326” without the quotes. The body of your email should contain the full public URL to the directory that contains the PHP and HTML files (make sure the directory is readable on the web). In addition, your email should have attached to it the .sql file that creates your database table.
